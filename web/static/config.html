配置

邮件通知

邮件服务器地址，如 smtp.qq.com 或 smtp.gmail.com
服务器端口，通常 SSL 为 465，非 SSL 为 25 或 587
您的邮箱账号，如 example@qq.com
邮箱授权码（非登录密码），需在邮箱设置中开启 SMTP 获取
是否启用加密，SSL/TLS 请选择 true，否则选择 false
发件人显示名称或邮箱地址
接收通知的邮箱，多个地址用英文逗号分隔

fnOS 系统连接

fnOS 的访问地址，需包含协议和端口，如 http://192.168.1.100:8000
fnOS 的超级管理员 Token，用于 API 鉴权
身份验证检查接口路径，默认一般无需修改
硬件信息获取接口路径，默认一般无需修改

性能与模式

硬件数据采集的频率，建议 60-900 秒
Static: 仅浏览已生成的报表; Dynamic: 实时调用 API
自动清理过期报表文件
限制应用使用的最大核心数

定时计划

每天自动发送静态报表的时间 (24小时制)

硬件 TDP 配置

自定义硬件功耗参数，用于更准确的功耗估算。留空则使用默认值。

CPU 的热设计功耗，通常 15-95W
硬盘待机时的功耗，通常 0.5-1.0W
硬盘读写时的功耗，通常 5-10W
SATA SSD 功耗，通常 2-4W
NVMe SSD 功耗，通常 3-6W
DDR4 或 DDR5 内存条功耗，通常 2.5-4W

Home Assistant 集成

配置 Home Assistant 集成接口。留空则不启用。
